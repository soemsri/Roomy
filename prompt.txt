5. อธิบาย Process การทำงานของอพาร์ทเม้นต์
 5.1 การ map LINE user กับระบบ
  5.1.1 ผู้เช่าแอด LINE OA
  5.1.2 ผู้เช่าทัก “สมัคร” หรือหมายเลขห้อง หรืออะไรก็ได้
  5.1.3 ระบบ webhook รับ → ได้ LINE userId
  5.1.4 มีระบบ backoffice ผ่าน Richmenu ใน LINE OA ให้เจ้าของกด map: LINE userId → ห้อง A101
 5.2 เวลาส่งบิลค่าเช่า
  5.2.1 backend คำนวณ หลังจากเจ้าของบันทึกมิเตอร์น้ำไฟประจำเดือนเข้าระบบ
  5.2.2 push message ไป userId ตามห้องที่เคย map ไว้
 5.3  ผู้เช่าดูค่าเช่า
  5.3.1 ผู้เช่ากดดูค่าเช่าจาก Richmenu บน LINE OA
  5.3.2 ผู้เช่าดูค่าเช่าผ่าน Richmenuในข้อ 5.3.1 จะพบบิลค่าเช่า บน web ที่ผูกกับ rihmenu ไว้ และมีปุ่มชำระค่าเช่าเป็นเงินสด หรือชำระด้วย QR ถ้าเลือกชำระค่าเช่าเป็นเงินสด ระบบจะแจ้งเป็น static ว่าให้ไปจ่ายที่หน้าเคาท์เตอร์ออฟฟิส พอผู้เช่าจ่ายค่าเช่าแล้วเจ้าของเซ็นต์ในบิลเพื่อยืนยันว่าจ่ายแล้วและถ่ายรูปบิลที่มีลายเช็นต์แล้วเจ้าของ Upload บิลที่มีลายเช็นต์นั้นผ่าน LINE ในส่วนของแอพเจ้าของเพื่อเก็บไว้เป็นหลักฐาน
  5.3.3 จากข้อ 5.3.2 ถ้าชำระค่าเช่าด้วย QR หลังจากผู้เช่าจ่ายเงินผ่าน QR Promptpay ให้เด้งเตือนเจ้าของว่ามีผู้เช่าจ่ายเงินแล้ว และระบบจ่ายผ่าน QR นี้เจ้าของสามารถ config ได้ว่าจะให้เปิดหรือปิดระบบการจ่ายเงินผ่าน QR นี้ได้ ถ้าเปิดไว้ผู้เช่าจะมองเห็นเมนูจ่ายเงินผ่าน QR แล้วขึ้น QR มาให้จ่ายเงินได้ แต่ถ้า config ให้ปิดการชำระด้วย QR ไว้ผู้เช่าจะไม่เห็นเมนูจ่ายผ่าน QR ซึ่ง QR Promptpay นี้มีให้ config ได้หลายบัญชีรับเงินด้วย
 5.4 แอพฝั่งของเจ้าของมีระบบ unmap ระหว่าง LINE userId กับห้อง เมื่อผู้เช่าย้ายออกหรือขอเปลี่ยนห้อง
 5.5 ผู้เช่าดูประวัติมิเตอร์น้ำไฟย้อนหลังได้ผ่าน Richmenu เจ้าของแก้ไขข้อมูลมิเตอร์น้ำไฟได้ถ้ายังไม่เลยจำนวนเดือน(ตั้งค่าได้)
 5.6 ผู้เช่าแจ้งซ่อมได้ผ่าน Richmenu โดย link เข้า web เพื่อให้กรอก เรื่องที่แจ้งซ่อม และแนบรูปถ่าย (Optional)
  - เมื่อแจ้งซ่อมแล้ว ให้ระบบส่ง Push Message แจ้งเตือนเจ้าของ (Admin) ทาง LINE ทันที
  - ในข้อความแจ้งเตือน Admin ให้แสดงรายละเอียดการแจ้งซ่อมและส่งรูปภาพที่แนบมาด้วย (ถ้ามี) ไปทาง LINE Admin Channel
 5.7 ระบบจัดการผู้อยู่อาศัย: ในหนึ่งห้องสามารถมีผู้อยู่อาศัยได้หลายคน โดยเจ้าของสามารถเพิ่มข้อมูลชื่อจริง, นามสกุล, ชื่อเล่น (บังคับกรอก), เบอร์โทรศัพท์ และสถานที่ทำงานของแต่ละคนได้ ข้อมูลเหล่านี้สามารถแก้ไขหรือลบได้ แต่ต้องมีรายชื่อผู้อยู่อาศัยค้างไว้อย่างน้อย 1 คนเสมอหากห้องนั้นมีการเชื่อมต่อ (map) กับ LINE user แล้ว
 5.8 ระบบประวัติผู้เช่า: เมื่อผู้เช่าย้ายออก (unmap) ระบบต้องทำการเก็บประวัติผู้อยู่อาศัยทั้งหมดที่เคยอยู่ในห้องนั้นไว้ และผู้ดูแลระบบสามารถค้นหาข้อมูลย้อนหลังได้จาก ชื่อ, นามสกุล, เบอร์โทรศัพท์ หรือสถานที่ทำงาน
 5.9 ระบบประวัติมิเตอร์: ผู้ดูแลระบบสามารถเรียกดูประวัติการบันทึกมิเตอร์น้ำและไฟฟ้าได้ตั้งแต่อดีตจนถึงปัจจุบัน โดยสามารถเลือกดูรวมทุกห้อง หรือเลือกดูแยกตามรายห้องได้
 5.10 ระบบประวัติการแจ้งซ่อม: ผู้ดูแลระบบสามารถเรียกดูประวัติการแจ้งซ่อมทั้งหมดได้ตั้งแต่อดีตจนถึงปัจจุบัน โดยสามารถดูรายงานรวมทั้งหมด หรือเลือกดูแยกเฉพาะรายห้องได้ และแสดงรูปภาพที่ผู้เช่าแนบมาในรายการแจ้งซ่อมบน Dashboard
 5.11 มีระบบ backoffice  ผ่าน Per-User Rich Menu ใน LINE OA ในส่วนของเจ้าของซึ่งเป็นเมนูที่เรา "สั่งเจาะจง" ให้แสดงผลเฉพาะ Admin เพื่อทำงานในส่วนต่างๆของเจ้าของ
 5.12 สำหรับ "ค่าใช้จ่ายอื่นๆ" ระบบปัจจุบันแบ่งการจัดการออกเป็น 2 รูปแบบตามประเภทการใช้งานครับ:

  5.12.1 ค่าใช้จ่ายที่ต้องเก็บ "ทุกเดือน" (เช่น ค่าส่วนกลาง, ค่าที่จอดรถ)
  รายการเหล่านี้จะถูกตั้งค่าเป็น "ค่าใช้จ่ายประจำ (Recurring Charges)" ซึ่งจัดการได้จาก 2 จุดในหน้า จัดการห้องพัก (Room Management) ครับ:
   * ตั้งค่าแยกรายห้อง: กดปุ่ม แก้ไขห้อง (ไอคอนดินสอ) ในตารางจัดการห้องพัก แล้วเพิ่มรายการในส่วน "ค่าใช้จ่ายประจำเดือน" ครับ
   * ตั้งค่าเหมือนกันทุกห้อง (เร็วที่สุด): กดปุ่ม "ตั้งค่าส่วนกลางทุกห้อง" ที่อยู่ด้านบนของตารางจัดการห้องพัก คุณสามารถใส่ "ค่าส่วนกลาง 100 บาท" แล้วกด
     Apply ระบบจะนำรายการนี้ไปใส่ไว้ในทุกห้องของอาคารนั้นให้ทันทีครับ
  > การทำงาน: เมื่อถึงเวลาคุณจดมิเตอร์ (ทั้งแบบรายห้องหรือรายอาคาร) ระบบจะไปดึงรายการเหล่านี้มาบวกรวมในบิลให้โดยอัตโนมัติครับ

  5.12.2 ค่าใช้จ่ายที่เก็บ "เฉพาะครั้ง" (เช่น ค่าปรับ, ค่าล้างแอร์, ค่าอุปกรณ์เสีย)
  รายการเหล่านี้จะจัดการจากหน้า "จดมิเตอร์น้ำไฟ" ในโหมด "จดทีละห้อง" ครับ:
   * เมื่อคุณเลือกเลขห้อง ระบบจะดึงค่าใช้จ่ายประจำ (จากข้อ 1) ขึ้นมาโชว์ก่อน
   * คุณสามารถกดปุ่ม "+ เพิ่มรายการ" เพื่อใส่รายการพิเศษเฉพาะของเดือนนั้นเข้าไปเพิ่มได้ครับ
  > การทำงาน: รายการพิเศษนี้จะปรากฏแค่ในบิลรอบเดือนนั้นเดือนเดียว ไม่ไปกระทบเดือนถัดไปครับ
